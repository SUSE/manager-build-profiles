This profile can be use to migrate from SUSE Manager Proxy 4.3 to MLM Proxy 5.0. It was tested with a re-instalation of proxy 5.0 on top of a machine with 4.3 proxy.

After the re-instalation finished, Users just need to generated the proxy configuration and deploy it.
Minions connected through the proxy should be able to recover their connection to the MLM server.
