# SUSE Multi-Linux Manager 5.0 Proxy Profiles (Legacy)

This directory contains AutoYaST profiles for deploying SUSE Multi-Linux Manager 5.0 Proxy and Retail Branch Server.

## Files included:
- `MLM_Proxy-50-SLES-Install.xml`: SLES installation for MLM 5.0 Proxy.
- `MLM_Proxy-50-SLMicro-Install.xml`: SL Micro installation for MLM 5.0 Proxy.
- `MLM_RBS-50-SLES-Upgrade.xml`: Upgrade profile for Retail Branch Server.
